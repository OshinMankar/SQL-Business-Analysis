# SQL-Business-Analysis

## Project Overview

This project analyzes chocolate sales data using SQL and MySQL to uncover insights related to revenue, profitability, product performance, customer demographics, and store operations.

The objective of this analysis is to transform raw transactional data into actionable business insights that can support strategic decision-making.

## Tools Used
- MySQL
- SQL
- VS Code
- GitHub

## Dataset Overview

The dataset consists of five tables:

- Sales (transaction-level data)
- Products
- Stores
- Customers
- Calendar

The Sales table serves as the fact table, while Products, Stores, Customers, and Calendar act as dimension tables for analytical reporting.

## Business Questions
- What is the overall business performance?
- Which product categories generate the highest revenue and profit?
- Which brands contribute most to sales?
- Which products are top performers?
- Which countries generate the highest revenue?
- Which store types perform best?
- What customer demographic patterns exist?

## Key Findings
### Executive KPIs

- Total Revenue: $154,543
- Total Profit: $61,637
- Profit Margin: 39.88%
- Total Orders: 6,034
- Total Customers: 5,681

### Product Performance
- Praline generated the highest revenue ($42,149).
- Praline also generated the highest profit ($16,761).
- Dark Chocolate 50% was the highest revenue-generating product.
- Revenue and profit trends were consistent across categories, indicating healthy pricing and cost structures.

### Brand Performance
- Cadbury was the highest revenue-generating brand.
- Revenue distribution across brands was relatively balanced, reducing dependency on a single brand.

### Store Performance
- Canada generated the highest revenue among all countries.
- Airport stores produced the highest revenue among store formats.
- Retail stores generated the lowest revenue.

### Customer Analysis
- Customer demographics were evenly distributed between male and female customers.
- Due to limited linkage between customer and sales records, customer demographic analysis was conducted independently from revenue analysis.

### Technical Skills Demonstrated
- Data Aggregation
- SQL Joins
- Group By Analysis
- Window Functions
- Business KPI Development
- Data Interpretation
- Business Recommendation Generation

## Conclusion

The analysis revealed that Praline products, Cadbury brand offerings, Canadian markets, and Airport stores are the strongest contributors to business performance. The business maintains a healthy profit margin while benefiting from a diversified product portfolio and balanced brand contribution.
